# (스킬 이름) — 수집 스킬

## 언제 작동하나
- 사용자가 [소스: 예) 유튜브 URL / 뉴스레터 / 캡처 이미지]를 주며 "수집/저장"을 요청할 때

## 무엇을 하나
1. 입력 소스를 읽는다
2. 마크다운으로 변환한다 (본문 + 핵심 정리)
3. frontmatter를 붙인다:
   ```yaml
   ---
   title: (제목)
   type: clipping
   source: (원본 URL/경로)
   created: (오늘 날짜)
   tags: []
   ---
   ```

## 어디에 저장하나
- `20_resources/` 아래에 `(소스명)_(날짜).md`로 저장
- 원문(raw)은 수정하지 않는다 — 가공 정리는 별도 노트로

## 안전
- 민감정보는 저장하지 않는다
- 저장 전 어디에 무엇을 만들지 한 줄로 보고
