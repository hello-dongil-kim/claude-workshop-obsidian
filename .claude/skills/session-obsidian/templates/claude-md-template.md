# (내 Vault 이름) — Claude 작업 규칙

## 이 Vault는
- 용도: 개인 지식 창고(두 번째 뇌)
- 구조(지도):
  - `00_inbox`: 받은함 / `01_projects`: 활성 / `20_resources`: 참고 / `90_templates`: 템플릿
  - 모든 노트는 frontmatter(`title`/`type`/`tags`/`created`) 포함

## 문체
- 한국어로 작성

## 파일명 규칙
- 개념·영구 노트: 제목 그대로 (예: `두 번째 뇌`)
- 자료성(클리핑·일지·리포트): `YYYY-MM-DD_제목`
- 공백 대신 `-`/`_`

## 작업 규칙
- 새 노트는 frontmatter 포함, 관련 개념은 `[[wikilink]]`로 연결
- 원문 클리핑(raw)은 수정하지 않는다 — 가공물은 별도 노트로
- `.obsidian/` 폴더는 절대 수정하지 않는다
- vault 전체 대량 수정 전에는 계획부터 보고

## 안전
- API key·토큰·개인정보는 읽거나 저장하지 않는다
- 최종 반영 전 변경 요약(diff)을 먼저 보여준다
