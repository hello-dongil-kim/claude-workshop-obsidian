# Block 4: 운영 ① 투입 루프 — Web Clipper로 지식 쌓기

## EXPLAIN

지금까지는 **구축**이었다. 이제 매일 굴리는 **운영**이다. 세팅보다 이 루프가 진짜 가치다.

### 투입 루프

```
[웹글] --Web Clipper--> [vault: 클리핑 노트(.md+frontmatter, 원문=불변)]
        --Claude Code--> [핵심주장 / 근거 / 활용 아이디어로 가공]
        --환원--> [vault에 정리노트 저장 + index·log 갱신]
        --재사용--> [다음 작업이 이 노트를 맥락으로 읽음 → 복리 누적]
```

### 입력은 한 가지가 아니다 — 다양한 수집 경로

Web Clipper는 **웹용 가장 쉬운 입구**일 뿐, vault에 MD를 넣는 길은 여러 개다. 소스가 무엇이든 "마크다운으로 바꿔 vault에 넣으면" Claude가 읽는다.

| 소스 | 수집 도구 | 결과 |
|---|---|---|
| 웹 기사 | **Web Clipper** | MD 클리핑 |
| PDF·PPTX | **pdf2md** (직접 만든 스킬) | 이미지 임베드 MD |
| Confluence 위키 | **wiki2md** (직접 만든 스킬) | 첨부 포함 MD |
| 회의 녹음(오디오) | **meeting-log** (직접 만든 스킬) | 회의록 MD |
| 손메모·대화 | 직접 작성 / Claude | MD |

> `wiki2md`·`pdf2md`·`meeting-log`는 누가 만들어준 게 아니라 **필요해서 직접 만든 스킬**이다. "이 소스를 자주 넣네 → 수집 스킬로 만들자"가 되는 순간 두 번째 뇌의 입구가 무한히 넓어진다. → **직접 만들어보는 건 선택 부록 B7에서.**

### Web Clipper — 가장 빠른 첫 입구

**Obsidian Web Clipper**는 웹 기사를 마크다운으로 변환해 vault에 저장하는 브라우저 확장이다. (Karpathy도 원전에서 직접 추천한 도구.)

- 설치 후 **저장할 Vault와 폴더**(예: `20_resources/clippings`)를 지정
- 팁: 이미지도 로컬에 받으려면 Obsidian 설정 → 파일/링크 → 첨부파일 폴더 지정

### 가공 = AI의 grunt work

클리핑은 **원문(raw)**일 뿐이다. 진짜 가치는 Claude가 가공할 때 나온다:

- 핵심 주장 / 근거(링크) / 반박 가능성 / 활용 아이디어로 나눠 정리
- 원문은 **건드리지 않고**, 가공 결과는 **별도 정리노트**로 만든다

### 질문 → 환원 (채팅에 흘려보내지 말 것)

질문해서 얻은 좋은 답변(비교표·분석·발견)도 **노트로 되돌려** vault에 쌓는다. 그래야 탐색 자체가 복리로 누적된다. 채팅 기록에만 남으면 사라진다.

## EXECUTE

다음을 직접 실행해보세요:

**Step 1: Web Clipper 설치 + 클리핑 1건**
- 브라우저에 "Obsidian Web Clipper" 확장 설치
- 설정에서 저장할 Vault와 폴더(예: `20_resources/clippings`) 지정
- 관심 있는 웹 기사 하나를 클리핑해 vault에 저장

**Step 2: Claude에게 가공 요청**
```
방금 클리핑한 이 노트를 읽고, 핵심 주장 / 근거 / 반박 가능성 / 활용 아이디어로 나눠
새 정리노트로 만들어줘. 원문 노트는 수정하지 말고.
```

**Step 3: 좋은 답을 환원**
- 정리 결과가 쓸 만하면 그대로 vault에 노트로 저장되었는지 확인

## QUIZ

**문제:** "투입 루프"에서 원문 클리핑 노트를 수정하지 않고 두는 이유는?

| 선택지 | 내용 |
|--------|------|
| A | 용량을 아끼려고 |
| B | 원문은 '진실의 원천'이라 불변으로 두고, 가공물은 별도 노트로 쌓기 위해 |
| C | Web Clipper가 수정을 막아서 |

**정답:** B — 원문(raw)은 건드리지 않는 진실의 원천으로 두고, AI가 만든 요약·정리는 별도 노트로 쌓는다. 그래야 나중에 원문으로 되돌아가 검증할 수 있고 지식이 안전하게 누적된다.
