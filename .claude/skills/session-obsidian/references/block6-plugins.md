# Block 6: 플러그인으로 더 나아가기 (선택)

## EXPLAIN

여기까지 왔으면 두 번째 뇌의 **핵심은 이미 완성**됐다. 플러그인은 필수가 아니다. 다만 **"AI가 읽는 DB"를 강화하는 것**만 골라 쓰면 vault가 더 강해진다.

> **고르는 기준 한 줄:** 많이 깔수록 좋은 게 아니다. **"이게 AI 협업(투입·구조·검색)을 강화하나?"** 로 고른다. 나머지는 필요해질 때.

### 자료 투입 — AI가 읽을 수 있게

| 플러그인 | 왜 두 번째 뇌에 | 활용 |
|---|---|---|
| **Text Extractor** | 이미지·스캔 PDF 속 글자를 OCR로 추출 → **그림 속 텍스트도 Claude가 읽음** | 캡처·손글씨·스캔 자료를 투입 전 텍스트화 |
| **PDF++** | PDF 주석·하이라이트를 노트로 연결 | 논문·리포트 소싱(B4 투입의 원문 단계) |

### 구조·자동화 — frontmatter를 눈으로

| 플러그인 | 왜 두 번째 뇌에 | 활용 |
|---|---|---|
| **Dataview** | frontmatter를 쿼리해 자동 목록·표(대시보드) → **"frontmatter=정밀 필터"를 눈으로 확인** | `type: clipping` 모아보기, 미정리 노트 자동 집계 |
| **Templater** | core 템플릿보다 강력(날짜·변수·스크립트) | 클리핑·정리노트 템플릿 자동 채우기 |
| **(core) Templates · Daily notes** | 기본 템플릿·일일노트 | 매일 들어오는 메모의 일관된 입구 |

> Dataview 쿼리 문법이 어렵다? **Claude에게 "이 조건으로 Dataview 쿼리 짜줘"** 하면 된다 — 또 하나의 협업 포인트.

### 검색·탐색 — 사람용 (Claude의 grep을 보완)

| 플러그인 | 왜 두 번째 뇌에 | 활용 |
|---|---|---|
| **Omnisearch** | vault 전체 본문 빠른 검색 | 사람이 직접 찾을 때 |
| **(core) 백링크** | 어떤 노트가 이 노트를 링크했나 | 그래프 탐색, 고아 노트 발견 |

### AI 작성 — Claude Code와 역할 비교

| 플러그인 | 왜 두 번째 뇌에 | 활용 |
|---|---|---|
| **Smart Composer** | Obsidian 안에서 AI 채팅·편집(Cursor류) | **가벼운 편집은 Obsidian 내장 AI, 대량·구조·파일 작업은 Claude Code** — 역할 분담 |

### 취향 — 주제와 약결합 (필요하면)

**Excalidraw**(손그림·다이어그램), **Google Calendar**(일정 연동), **Typewriter Mode**(집중 글쓰기). 두 번째 뇌의 핵심은 아니지만 작업 취향에 맞으면.

**Step 1: 커뮤니티 플러그인 켜기 + 1개 설치**
- 설정 → 커뮤니티 플러그인 → "제한 모드 끄기" → "찾아보기"
- 추천 1개 설치: **Omnisearch**(검색) 또는 **Dataview**(구조)

**Step 2: Dataview를 깔았다면, 쿼리는 Claude에게**
- 노트 하나에 아래를 부탁해서 만든 쿼리를 붙여넣어 본다:
```
내 vault에서 frontmatter type이 clipping인 노트를 표로 보여주는 Dataview 쿼리를 만들어줘. 컬럼은 제목, 생성일, source.
```
- (또는) Text Extractor를 깔았다면 이미지 1개에서 텍스트를 뽑아본다

## QUIZ

**문제:** 이 세션 관점에서 플러그인을 고르는 기준은?

| 선택지 | 내용 |
|--------|------|
| A | 유명한 것부터 일단 다 깐다 |
| B | "AI가 읽는 DB(두 번째 뇌)를 강화하는가"로 고른다 |
| C | 많이 깔수록 vault가 좋아진다 |

**정답:** B — 플러그인은 많이 깐다고 좋아지지 않는다. 자료 투입(Text Extractor·PDF++), 구조(Dataview·Templater), 검색(Omnisearch)처럼 *AI 협업을 강화하는지*로 고르고, 나머지는 필요할 때 더한다.
